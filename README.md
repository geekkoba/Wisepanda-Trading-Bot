# wisepanda-trading-bot

### Create a Virtual Environment

```shell script
python -m venv venv
```

### Activate the Virtual Environment

On Windows:

```shell script
venv\Scripts\activate
```

On MacOS and Linux:

```shell script
source venv/bin/activate
```

### Install Packages

```shell script
pip install -r requirements.txt
```

### Start Application

```shell script
python main.py
```
